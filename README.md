Star Ruby 0.4.0 (2010-06-29)
=======

## About Star Ruby

Star Ruby is a ruby extensional library for creating games.
You can develop SNES-like 2D games with Star Ruby.


## Dependent Libraries

  - Fontconfig (Optional)
  - libfreetype
  - libGL / libopengl32
  - libpng
  - libogg
  - libruby18
  - libvorbis
  - libvorbisfile
  - SDL
  - SDL_mixer
  - SDL_ttf
  - SMPEG
  - zlib


## How to Install

  $ ruby extconf.rb [--with-opt-dir=DIRECTORY]
  $ make
  $ make install


## How to Do Tests

  $ ruby -Ctest -I.. runner.rb
