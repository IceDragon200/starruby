* About Star Ruby

Star Ruby is a ruby extensional library for creating games.
You can develop SNES-like 2D games with Star Ruby.


* How to Install

Install Ruby %ruby_version% if you need.

  Active Script Ruby (Windows Installer)
  http://arton.no-ip.info/data/asr/

Execute "install.rb" to install Star Ruby.

  > ruby install.rb
